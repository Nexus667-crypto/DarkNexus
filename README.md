# Ban-All Bot

Bot Discord avec la commande `/ban-all` qui bannit tout le monde sauf toi et les bots.

## Installation

1. Crée une application sur https://discord.com/developers/applications
2. Active les 3 Privileged Intents (Server Members Intent, Presence Intent, Message Content Intent)
3. Récupère le Token et ton ID Discord
4. Déploie sur Railway en ajoutant les variables `TOKEN` et `OWNER_ID`
